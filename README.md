# ToMom门店管理系统



## 环境准备

安装 `node_modules`:

```bash
npm install
```

or

```bash
yarn
```

## 目前提供的npm命令

命令在package.json可以查看，这里仅罗列主要的命令

### 运行项目命令：

```bash
npm start
```

### 项目打包命令：

```bash
npm run build
```

### 代码格式化检查：

```bash
npm run lint
```

You can also use script to auto fix some lint error:

```bash
npm run lint:fix
```

### 代码测试

```bash
npm test
```
